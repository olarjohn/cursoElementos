# Elementos de la teoria - Curso

Material interactivo en HTML para temas de matematica discreta:

- Congruencias y congruencias lineales
- Conjunto Z/nZ
- Induccion matematica y estructuras inductivas

## Estructura del proyecto

```text
curso/
  Elementos.html
  curso4.html
```

## Como usar

1. Abre `curso/Elementos.html` en tu navegador para ver la pagina de congruencias.
2. Abre `curso/curso4.html` en tu navegador para ver la pagina de induccion matematica.

Tambien puedes abrir la carpeta en VS Code y usar una extension como Live Server para previsualizacion local.

## Contenido de las paginas

### Elementos.html

Incluye:

- Definicion de congruencia
- Ejemplos y ejercicios con solucion desplegable
- Congruencias lineales
- Operaciones en Z/nZ

### curso4.html

Incluye:

- Explicacion del principio de induccion
- Visualizacion por bloques (caso base, suposicion, paso inductivo)
- Ejercicios guiados
- Editor en vivo con soporte LaTeX (MathJax)
- Modo paso a paso

## Requisitos

No requiere instalacion de dependencias. Solo un navegador web moderno.

Si usas `curso4.html`, se recomienda conexion a internet para cargar MathJax desde CDN.
